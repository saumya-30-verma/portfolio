## ⚙️ Installation & Setup

Clone the repository:

```
git clone https://github.com/your-username/portfolio.git
```

Navigate to the project directory:

```
cd portfolio
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```
